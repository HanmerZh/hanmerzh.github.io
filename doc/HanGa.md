# 汉格：用系统功能语言学重构你的多模态知识笔记
汉格的诞生，是一场关于语言学的数字人文实验。在立项初期，它的名字叫“HanGarden”，寓意“妙笔生花，汇聚成园”。但随着研发的深入，我们决定将其定名为“汉格（HanGa）”——它不再仅仅是一个知识的园林，而是一个能承载个体意义的结构化框架。

## 灵感与系统功能语言学的契合

在语言学界，系统功能语言学由韩礼德创立。他认为语言不仅是形式的集合，更是社会行为的工具，是意义的潜势。简单来说，我们在记录生活时，其实是在特定的语境下，从一套复杂的潜势系统中进行主观选择。汉格的设计初衷，就是希望在数字化的浪潮中，为你梳理一套以主题为中心的个人多模态资源库，从而帮助你捕捉那些真正重要的经验。

汉格的核心逻辑，是将三大元功能转化为触手可及的数字化体验：

其一，概念功能：文字在再现真实经验时，往往存在描述力的乏力。汉格希望你可以围绕一个主题同时记录视频、音频、地理坐标等信息，重构一个立体的记录空间。并且进行精美再现。

其二，人际功能：通过录音与视频模块，这些多模态数据让笔记不再是冰冷的数据，而是一段有温度且指向自我的对话。

其三，语篇功能：汉格通过结构化的框架，引导你将零散的灵感、即时的语音或影像，组织成连贯、有条理的数字语篇。它希望能解决你在数字化时代信息过载却意义缺失的问题，将杂乱的数据转化为结构化的语篇。

希望每一位汉格的使用者，都能在这套结构化框架中，重新发现记录的快乐，并找回那些被碎片化时代所消解的意义细节。
