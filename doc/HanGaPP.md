# 汉格隐私保护政策

汉格是一款基于系统功能语言学理论打造的本地化多模态笔记工具。我们深知个人信息及思维记录对您的重要性，并庄严承诺保护您的隐私安全。

本政策旨在说明汉格在本地化运行过程中，如何处理您的相关信息：

本地化运行：本产品核心功能均在您的设备本地完成。

数据所有权：您数据（文本、图像、音视频等）存储于您的本地沙盒环境。

AI说明：本应用程序现阶段并不提供任何AI功能。

# 一、我们如何收集和使用信息

汉格遵循“合法、必要”原则，出于以下工具化目的处理信息：

本地分析：当您导入数据时，汉格会在本地扫描文件属性（如分辨率、时长、文本编码）。

当您保存数据时，我们需要获得存储访问权限以完成文件写入。

地理坐标信息：当您在笔记中标记位置或使用地图功能时，汉格会调用设备位置信息（需要网络及位置访问权限）。该信息仅用于在本地笔记中建立地图信息，不会上传至我们的服务器。

# 二、设备权限调用说明

为了实现多模态数据的录入，汉格可能需要申请以下权限 ：

相册/文件访问 (Storage)：用于导入图片、视频文件，以及导出相关数据。

麦克风 (Microphone)：用于录入音频文件。

语音识别 (Speech Recognition)：用于将音频数据转化为文本。

摄像头 (Camera)：用于现场拍摄影像素材或扫描相关元数据信息。

地理位置 (Location)：用于在笔记中标记坐标。

网络（Network）：地理位置等部分功能需要联网使用。

您可以随时在系统设置中撤回这些授权，这不会影响您使用本产品的其他核心功能。

# 三、数据的存储与安全

存储位置：汉格是一个本地化运行的应用。除非您主动选择分享，否则您的所有项目文件数据均存储在您的 iOS 设备本地。

安全机制：我们利用 iOS 系统级的沙盒隔离机制保护您的本地数据。

无社交分发：本产品不设社区或社交广场，您的数据不会被第三方或本产品其他用户获取。

# 四、您的权利：管理与删除

项目删除：您可以随时在 App 内删除特定的数据项目。一旦执行，本地相关的所有缓存及处理文件将永久移除。

安全建议：尽管汉格是本地存储，但如果您在提示词中包含敏感个人信息，请务必保护好您的设备访问密码。

# 五、隐私政策的更新

随着汉格功能的迭代，我们可能会适时修订本政策。重大变更将通过应用内显著提醒的方式告知。
